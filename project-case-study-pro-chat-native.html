Pro Chat Native

A native chat experience built with Expo (React Native) using Firebase Auth and Firestore for real-time messaging. GiftedChat provides the message UI while Expo Router handles navigation across iOS, Android, and web.

Tech

  • Expo / React Native
  • Expo Router
  • Firebase Auth & Firestore
  • react-native-gifted-chat
  • TypeScript / JavaScript

Notes

This project demonstrates real-time messaging with Firestore snapshot listeners and a native-quality UI using GiftedChat. The repository contains inline Firebase configuration for quick testing; consider moving secrets to environment variables or a private config when publishing.
